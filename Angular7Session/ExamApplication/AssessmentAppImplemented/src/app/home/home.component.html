Quick Tips

  • Adding Jquery : make entry in angular.json file in script [] as "node_modules/jquery/dist/jquery.js"
  • Including Bootstrap js file : make entry in angular.json file in script [] as "node_modules/jquery/dist/js/bootstrap.js"
  • Including Bootstrap css file : make entry in style.css @import "~bootstrap/dist/css/bootstrap.css";
  • Use API for GET/POST/DELETE verbs https://jsonplaceholder.typicode.com/users
